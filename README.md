# template-e-comerce-ui

Plantilla de documentacion arquitectonica y de gestion del proyecto
para el frontend e-commerce **PracticaYoruba UI**.

Este repositorio extrae como template reusable la estructura producida
por la iniciativa `analizar-ramas-pendientes-de-integracion` ejecutada
sobre `jcg-admin/e-comerce-ui` el 2026-05-20.

## Que hay aqui

| Carpeta | Contenido |
|---------|-----------|
| `docs/` | Documentacion arc42 adaptada (sin numeros, sin emojis) + modulo `pm/` con iniciativas segun PROC-GESTION-001. |

## Convenciones aplicadas

- Markdown puro (sin Sphinx, sin RST).
- Diagramas en mermaid embebido (renderizable nativamente en GitHub).
- Sin numeros en nombres de carpetas ni archivos.
- Slugs autoexplicativos.
- Sin emojis ni iconos.
