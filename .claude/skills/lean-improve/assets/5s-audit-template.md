## Escala de evaluación

| Puntaje | Descripción |
|---------|-------------|
| **1** | Inexistente — la S no ha sido implementada o se abandonó completamente |
| **2** | Inicial — implementación parcial; muchas desviaciones visibles |
| **3** | En progreso — implementación visible pero inconsistente; algunas desviaciones |
| **4** | Bueno — implementación consistente; desviaciones menores ocasionales |
| **5** | Excelente — estándar mantenido consistentemente; es el nuevo normal del área |
