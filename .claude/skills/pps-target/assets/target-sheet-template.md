> Reducir/Mejorar [métrica] de **[baseline]** a **[meta]** para **[fecha]**,

> sin degradar [métrica secundaria] por debajo de [umbral].

**Redacción del target:**

[Completar con los valores específicos del proyecto]

**Verificación SMART:**

| Criterio | Verificación | ✅ / ❌ |

| **Específico** | ¿La métrica es exacta y unívoca? | |
| **Medible** | ¿El método de medición está definido? | |
